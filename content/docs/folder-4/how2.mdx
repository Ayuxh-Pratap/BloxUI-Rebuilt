## CLI Components

### Installation and Setup

To get started with the BloxUI CLI Tool, follow these steps:

#### Step 1: Install Node.js

Ensure that Node.js is installed on your system. You can download it from [Node.js official website](https://nodejs.org/).

#### Step 2: Install the CLI Tool

Open your terminal and run the following command to install the BloxUI CLI Tool:

```bash
npx bloxui-comadd init
```

#### Step 3: Add Components

Congratulations , now are free to add any component from BloxUI's `CLI Components` ,

You will get the sepecific command for each component in its folder.
